# EDA6-Google_Play_Store_Apps

Kaggle Link:- https://www.kaggle.com/code/archissave/eda-6-google-play-store-apps

Many columns have missing or inconsistent values. Size column has mixed formats like MB, KB, or “Varies with device”. Installs column contains symbols like “+” and “,” that need cleaning. Price column sometimes has invalid entries mixed in. Most apps target “Everyone” as content rating.Productivity, Lifestyle, and Communication are also popular categories. Only a small fraction of apps have ratings above 4.5. A few popular apps dominate download counts, while most apps have under 1M installs. Free apps rely heavily on ads and in-app purchases for revenue.
